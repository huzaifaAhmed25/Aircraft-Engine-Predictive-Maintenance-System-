# Aircraft-Engine-Predictive-Maintenance-System-
In this project, I developed an AI-powered system that predicts the Remaining Useful Life (RUL) of aircraft engines using real-world sensor data.
🔍 Instead of relying on fixed formulas, the model learns patterns from:
• Temperature 🌡️
• Pressure ⚙️
• Operational conditions
• Sensor trends over time
Using Machine Learning (XGBoost) and advanced feature engineering, the system can:
✔ Predict engine failure before it happens
✔ Monitor real-time engine health
✔ Simulate live performance with dynamic visualization
✔ Provide explainability using SHAP (why the model predicts failure)

💡 One key highlight:
This is a data-driven predictive maintenance system, not a traditional physics-based model — making it scalable for real-world industrial use.
📊 Built with:
• Python
• Streamlit (interactive dashboard)
• XGBoost
• SHAP (Explainable AI)
